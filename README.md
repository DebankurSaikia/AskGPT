# AskGPT

AI-powered chat application built using MERN Stack, JWT Authentication, MongoDB Atlas, and Google Gemini API.

## Features
- User Authentication (Login / Signup)
- JWT Protected Routes
- Multiple Chat Threads
- Delete Threads
- Markdown Rendering
- Syntax Highlighting
- Responsive UI
- Chat History Persistence

## Tech Stack
- React
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Google Gemini API
